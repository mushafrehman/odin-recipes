# odin-recipes

## Description
This is the first portfolio project from [The Odin Project's](https://www.theodinproject.com/) Foundations course. It is a basic recipe website built to practice fundamental web development skills, focusing primarily on HTML structure and version control.

## What I Have Learned So Far
Building this project has allowed me to put theory into practice. Here are the core concepts I have applied:
* **HTML Basics:** Understanding how to structure a webpage using proper elements and tags.
* **The Boilerplate:** Setting up the foundational `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>` tags.
* **Working with Media and Navigation:** Successfully embedding images and creating functional links to connect different pages.
* **Version Control:** Using Git for tracking changes and writing clear, descriptive commit messages.
* **Introduction to CSS:** Getting a glimpse into how styling will eventually be applied to HTML structures.

## What I Will Achieve by the End of this Project
By completing this recipe website, I will have:
* Built a complete, multi-page website entirely from scratch.
* Solidified my local development workflow using the command line, Git, and GitHub.
* Gained the confidence to take a blank text file and turn it into a working webpage that displays nicely in a browser.
